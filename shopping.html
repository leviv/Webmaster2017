Holo Lens

Software

Microsoft’s Hololens is the longest-standing piece of mixed reality application technology. It runs on Windows 10 and has an array of sensors to allow for different movements in order to control the augmented reality. The main types of movements are Gaze and Gesture. Gaze manipulates the cursor to be pointed in the direction that the user moves their head, and Gesture allows for native hand movements to manipulate objects by placing and sizing them. There are also voice commands that link directly with Cortana for maximum flexibility within the program. The Hololens also incorporates collaborative workspaces with Skype integration.

Hardware

TThe Hololens is built to allow the user maximum comfort so they can use it for several hours in a row. It has a particularly small screen, causing the field of vision to be somewhat limited, but it has 2GB of RAM and 64GB of flash memory to make up for this. The camera on the Hololens is 2.4 megapixels.

Meta 2

Software

Meta 2 is the latest in Meta’s headset development. It utilizes a software they call the “Neural Path of Least Resistance” that they collaborated with neuroscientists to create. This software allows the wearer to use gestures that are similar to the natural way we interact with the real world in order to interact with the objects projected. The sensors in the headset allow one’s hands to be the main tool, even in the collaborative workspaces that the Meta 2 also incorporates. As of now, it only runs on Windows 8 and 10, but there is Mac development planned for later in 2017.

Hardware

Right now, the Meta 2 is early in its developer beta, so there aren't so many technical specs available to the public. The camera is a mere 720p, but the screen is 2k resolution and has a large screen from the eyebrows to the base of the nose in order to allow for the widest field of vision. Developers who try the Meta 2 still have to plug the device into their computer.

Bridge

Software

The Bridge is a wearable headset by Occipital that uses your iPhone as the computer. Therefore, it runs on iOS 9. When it is plugged into the phone, it enables obstacle avoidance so that the user can freely move around and be notified if they are about to encounter objects. It has six degrees of freedom tracking for this feature and for the user’s movement around the world.

Hardware

Since the Bridge is optimized for iPhone, the hardware specs vary from phone to phone. It will have the same resolution, storage, and camera specs as the phone plugged into it.

Magic Leap

Software

The Magic Leap is the newest concept on this list. From the early demo videos, it is known that the headset is more aimed towards gamers than other headsets. Magic Leap has incorporated natural gestures in order to interact with the environment, and allows for not only objects but information to be placed around the room in one’s field of vision, like on a computer screen. This information can be interacted with in the same way as an object in the system.

Hardware

Since Magic Leap has yet to even release a beta, the public knows next to nothing about the technical specs. However, it is similar to the Meta 2 with the size of the screen and the fact that it does have to be plugged into the computer.
